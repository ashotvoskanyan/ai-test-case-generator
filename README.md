# AI Test Case Generator

AI Test Case Generator is a beginner project for learning Manual QA, AI tools, and basic software workflows.

## What this project does

This project takes a user story or product requirement and turns it into QA testing materials.

## Input

A user story or product requirement.

## Output

- Test scenarios
- Positive test cases
- Negative test cases
- Edge cases
- Clarifying questions
